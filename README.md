## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/hectic-labs.git
```

Navigate into the project:

```bash
cd hectic-labs
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
